Settings · Apps Script 연결

Google Apps Script 웹앱 URL과 Secret Key를 저장하면 이 앱이 Google Sheets에 어린이, 교사, 출석, 헌금 데이터를 저장하고 불러올 수 있습니다.

Settings

관리자 PIN

Secret Key는 Apps Script에도 같은 값으로 넣어야 합니다. URL만 아는 사람이 접근하는 것을 줄이기 위한 간단한 보호 장치입니다.

Google Sheets 연동

Apps Script 웹앱 URL을 입력하세요.
형식: https://script.google.com/macros/s/.../exec

연결 테스트

데이터 동기화

데이터 초기화

앱 로컬 데이터를 초기화합니다. Google Sheets 데이터는 유지됩니다.

Apps Script 연결 순서
  1. Google Sheet → Extensions → Apps Script를 엽니다.
  2. Code.gs 내용을 붙여넣고 setup 함수를 한 번 실행합니다.
  3. Deploy → New deployment → Web app으로 배포합니다.
  4. 생성된 /exec URL을 위 Google Sheets 연동 칸에 붙여넣습니다.
  5. Secret Key는 Apps Script의 SECRET_KEY 값과 앱의 Secret Key가 같아야 합니다.
  6. 저장 후 연결 테스트를 누르세요.
